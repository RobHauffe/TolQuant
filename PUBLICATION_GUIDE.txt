## TolQuant v1.0.0

### 🎉 Initial Release

TolQuant is a scientific software application for analyzing glucose tolerance 
tests (GTT) and insulin tolerance tests (ITT) with advanced statistical methods.

### ✨ Features
- GTT and ITT data analysis
- Two-way repeated measures ANOVA
- Automatic post-hoc tests
- Interactive visualization
- Excel export with multiple sheets
- Standalone Windows executable

### 📦 Installation
Download TolQuant.exe from the Assets section below.

### 📖 Documentation
See README.md for detailed usage instructions.
